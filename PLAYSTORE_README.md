# ChelureTech - Play Store Listing

**Short Description (80 chars max):**  
View your IP address, network status, and device info. Simple and secure.

**Full Description:**

ChelureTech helps you quickly view your network and device information.

## Features

**Network Status**
- View your current IP address
- See connection type (WiFi, Mobile Data, Ethernet, Bluetooth, VPN)
- Check internet reachability status
- Monitor connection status (Online/Offline)

**Quick Actions**
- Refresh network information
- Copy IP address to clipboard
- Share IP address and connection details

**Device & Connection Info**
- IP address
- Connection type
- Airplane mode status
- Device brand and model
- Operating system and version
- Device name

**Settings**
- Dark mode toggle
- Auto refresh option
- Rate the app

**Privacy**
- Your device information is not shared with any third parties
